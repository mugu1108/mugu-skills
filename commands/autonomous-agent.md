# /autonomous-agent - 自律型エージェント構築

Claude Code を定期実行・自動監視する自律型エージェントとして設定する。

## 使用方法

```
/autonomous-agent [セットアップ|タスク追加|状態確認]
```

## いつ使うか

- 定期的に自動実行するタスクを設定したい時（PRレビュー、監視等）
- セッション跨ぎのタスクキューを管理したい時
