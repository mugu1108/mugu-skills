# /workspace-audit - Claude Code 環境診断

現在の Claude Code 環境（リポ構成、MCP、プラグイン、hooks）を診断し、最適化を提案する。

## 使用方法

```
/workspace-audit
```

## いつ使うか

- Claude Code のセットアップ後に環境を最適化したい時
- 新しいプロジェクトで活用できる機能を確認したい時
