# /slide - Marpスライド生成コマンド

**説明**: Marp形式のプレゼンテーションスライドを自動生成します。

## 使用方法

```
/slide [スライドの内容やトピック]
```

## 概要

`/slide` コマンドは、Slideスキルを呼び出し、Marp形式のプレゼンテーションスライドを生成します。用途に応じたテンプレートを選択し、デザインガイドラインに準拠したスライドを作成します。

## テンプレート選択

| テンプレート | キーワード |
|-------------|-----------|
| general-presentation | セミナー、登壇、カンファレンス |
| hands-on-workshop | ハンズオン、実習、ワークショップ |
| product-intro | 製品、サービス、ツール紹介 |

## 使用例

```
/slide Next.js 15の新機能についてのセミナー資料。対象は中級者、30分。
/slide TypeScript入門ワークショップ。初心者向け、2時間ハンズオン。
/slide Supabaseの製品紹介スライド。15分。
```
