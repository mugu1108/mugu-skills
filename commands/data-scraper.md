# /data-scraper - データ収集エージェント構築

無料スタック（Python + Gemini Flash + GitHub Actions）で定期データ収集エージェントを構築する。

## 使用方法

```
/data-scraper [収集対象の説明]
```

## いつ使うか

- WebサイトやAPIからの定期的なデータ収集を自動化したい時
- 求人、価格、ニュース等の監視を設定したい時
