# mugu-skills

Claude Code 用のスキル・コマンド・エージェント集。

## セットアップ

### スタンドアロン利用

```bash
git clone https://github.com/mugu1108/mugu-skills.git
```

### サブモジュールとして利用

```bash
git submodule add https://github.com/mugu1108/mugu-skills.git mugu-skills
```

## スキル一覧

| コマンド | 用途 |
|---------|------|
| `/code-review` | コード品質チェック |
| `/slide` | Marpスライド生成 |
| `/security-review` | セキュリティ監査 |
| `/doc-update` | docs/ 仕様書の自動同期 |
| `/md-review` | MD品質チェック |
| `/time-track` | 時間追跡 |
| `/expense-track` | 経費記録 |
