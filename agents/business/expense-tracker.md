# Expense Tracker Agent - 経費記録エージェント

## 概要

Expense Trackerエージェントは、レシート画像やテキスト入力からAIで経費情報を自動解析し、freee会計に記録するSlack Botエージェントです。青色申告（インボイス未登録）のフリーランス向けに設計されています。

> OCR + 勘定科目判定を行い、確認ボタンで人間がチェックしてから記録する「半自動」設計

税務相談機能も内蔵。`TAX_CHANNEL_ID` で指定したチャンネルに質問を投稿すると、shinkoku OSSリファレンス（32ファイル）に基づいて自動回答します。

## 使用タイミング

1. **レシート画像の投稿**: Slackにレシート画像を投稿すると自動でOCR解析
2. **テキスト経費入力**: `add` コマンドでテキストから経費を推定
3. **経費記録の確認**: スプレッドシートリンクの取得

## Slackコマンド

### レシート画像投稿

レシート画像をメンション付きで投稿すると自動解析します。

```
@expensebot [レシート画像]
```

### `add` - テキストから経費登録

```
@expensebot add スタバ コーヒー 550円
@expensebot add Amazon 技術書 3200円
@expensebot add 電車 渋谷→新宿 200円
```

### `summary` - スプレッドシートリンク

```
@expensebot summary
```

### `help` - ヘルプ表示

```
@expensebot help
```

## 処理フロー

```
レシート画像 or テキスト
    ↓
Gemini Flash API で解析
    ↓
確認メッセージ（Block Kit）
  ├── ✅ 保存 → Google Sheets に追記
  ├── ✏️ 修正 → スレッドで部分修正 → 再確認
  └── ❌ キャンセル → 破棄
```

## 勘定科目判定（11種類）

| 勘定科目 | 具体例 |
|---------|--------|
| 旅費交通費 | 電車・バス・タクシー・飛行機 |
| 通信費 | 携帯・インターネット・サーバー |
| 消耗品費 | 文房具・PC周辺機器（10万円未満） |
| 新聞図書費 | 書籍・技術書・学習サブスク |
| 接待交際費 | 飲食（取引先）・贈答品 |
| 地代家賃 | 事務所家賃・コワーキング |
| 水道光熱費 | 電気・ガス・水道 |
| 外注費 | 業務委託・デザイン発注 |
| 広告宣伝費 | Web広告・名刺・チラシ |
| 租税公課 | 印紙税・住民税 |
| 雑費 | 上記に分類できないもの |

## 修正フロー

スレッド返信で以下の形式で部分修正が可能:

```
科目 通信費
金額 1500
摘要 Amazon Web Services
支払 カード
日付 2026-02-15
税 8%対象（軽減）
```

## スプレッドシートカラム

| 列 | 内容 |
|----|------|
| A: 日付 | YYYY-MM-DD |
| B: 勘定科目 | 自動判定 |
| C: 金額（税込） | 整数 |
| D: 摘要 | 店名 + 内容 |
| E: 支払方法 | 現金/カード/電子マネー/振込 |
| F: 税区分 | 10%対象/8%対象（軽減）/非課税 |
| G: 登録日時 | JST |
| H: 備考 | ユーザーメモ |
| I: レシート画像 | SlackファイルURL |

## 環境変数

```bash
EXPENSE_BOT_TOKEN=xoxb-...      # Slack Bot Token
EXPENSE_APP_TOKEN=xapp-...      # Slack App Token
EXPENSE_CHANNEL_ID=C...         # 経費記録チャンネル
OPENAI_API_KEY=sk-...           # OpenAI API Key
TAX_CHANNEL_ID=C...             # 税務相談チャンネル（省略可）
TAX_REFERENCE_DIR=...           # リファレンスパス（デフォルト: ~/.agents/skills/tax-advisor/reference）
```

## 起動方法

```bash
npm run expense-tracker
```

## 税務相談機能

`TAX_CHANNEL_ID` が設定されている場合、指定チャンネルへのテキスト投稿に対して自動回答します。

処理フロー:
1. LLM（gpt-4o-mini）で関連リファレンスファイルを3-5件選択
2. 選択ファイルを読み込み、回答を生成
3. 「回答を準備中...」を回答で上書き表示

スレッド内の追加質問にも対応します。

## モデル選択

- **gpt-4o-mini**: レシートOCR・勘定科目判定・税務相談回答
