# /lp - クリップボード長文ペースト

**説明**: クリップボードの長文テキストをペースト制限なしで Claude Code に入力する。

## 使用方法

```
[指示内容] /lp
```

## 概要

Claude Code のチャット入力には文字数制限があり、長文をペーストするとバグや固まりの原因になる。`/lp` は `pbpaste` コマンドでクリップボードの内容を直接取得し、制限なしで処理できるようにする。

## ワークフローステップ

### ステップ1: クリップボード取得

`pbpaste` でクリップボードの内容を取得し、文字数と行数を確認する:

```bash
pbpaste
pbpaste | wc -c && pbpaste | wc -l
```

### ステップ2: 内容の確認

取得した内容のサマリーを表示:

```
クリップボード取得完了: 約X,XXX文字 / XX行
```

### ステップ3: ユーザー指示の実行

ユーザーが `/lp` と一緒に入力した指示に従って、クリップボードの内容を処理する。指示がない場合は内容を表示して次の指示を待つ。

## 使用例

### 例1: クリップボードの内容でファイル作成

```
以下の内容をもとに紹介文を書き直してmdファイルとして保存して /lp
```

### 例2: クリップボードの内容を分析

```
以下のエラーログを分析して原因を教えて /lp
```

### 例3: 指示なしで内容確認

```
/lp
```

→ クリップボードの内容と文字数を表示し、次の指示を待つ

## 注意事項

- macOS の `pbpaste` を使用（Linux では `xclip -selection clipboard -o` に置き換え）
- バイナリデータがクリップボードにある場合は正しく取得できない
