# Slideテンプレート

## テンプレート一覧

### general-presentation.md
一般的な登壇・セミナー用。問題提起 → 解決策 → 技術詳細 → 事例 → まとめの構成。
推奨時間: 20-60分。

### hands-on-workshop.md
ハンズオン形式ワークショップ用。理論30%/実習70%の構成で、`hands-on`/`checkpoint`/`challenge`マーカー付き。
推奨時間: 1-4時間。

### product-intro.md
製品・サービス紹介用。課題 → ソリューション → 機能 → デモ → ユースケース → CTA の構成。
推奨時間: 15-30分。

## テンプレート選択フローチャート

```
参加者が実際に手を動かす？
├─ はい → hands-on-workshop.md
└─ いいえ → 製品・サービスを紹介する？
           ├─ はい → product-intro.md
           └─ いいえ → general-presentation.md
```

### キーワードベースの選択

| テンプレート | キーワード |
|-------------|-----------|
| hands-on-workshop | ハンズオン, 実習, ワークショップ, hands-on |
| product-intro | 製品, サービス, ツール紹介, デモ, 機能紹介 |
| general-presentation | セミナー, 登壇, カンファレンス, 勉強会, 解説 |
